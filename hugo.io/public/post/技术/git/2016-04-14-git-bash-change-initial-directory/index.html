Git Bash 更改初始目录

Thu, Apr 14, 2016

每次打开Git Bash时，默认的初始目录是/c/Windows/system,切换工作目录还需要手动敲cd /../..，太过麻烦。

右键点击Git Bash启动项，打开属性，找到快捷方式的起始位置一栏，改为自己的工作目录路径就可以了。
